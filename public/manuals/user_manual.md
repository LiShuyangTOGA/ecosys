# 生态系统模拟器 · 用户手册

欢迎！这是一个纯前端的生态系统模拟游戏，你可以绘制自己的食物网，调整参数，然后观察种群随时间如何变化。

## 快速上手

1. **添加物种**：点击屏幕底部预设生物类别栏（🌿生产者 / 🐰草食动物 / 🦊肉食动物 / 🐻杂食动物），再点击上方出现的具体物种，物种节点会出现在画布中。
2. **连接食物链**：点击右侧工具栏的 ➡️ 连线按钮，然后**先点击猎物，再点击捕食者**，形成箭头连线。
3. **调整参数**：点击 ✏️ 调参工具，再点击节点或连线，弹出参数浮窗。可修改生物名称、体重、初始生物量，或开启“自动调参”由体重自动推算其他值。
4. **全局环境**：点击右下角 🌍 按钮，设置营养物质输入、分解速率等环境条件，也可选择预设场景（湖泊、草原等）。
5. **开始模拟**：点击右上角 ▶️ 按钮，节点上的数字会实时变动。再次点击 ⏸️ 可暂停，修改参数后重新开始。
6. **观察结果**：通过左侧导航栏切换页面，查看：
   - 📈 **个体数变化**：折线图，每种生物的数量随时间变化
   - ⚡ **种群总储能**：折线图，能量变化
   - 🔺 **生态金字塔**：拖动滑块查看任意时刻的生物量/能量/数量金字塔

## 重要按钮说明

| 按钮 | 功能 |
|------|------|
| 📘 | 打开本文档（帮助） |
| 💾 | 保存当前食物网为 JSON 文件 |
| 📂 | 上传之前保存的食物网 JSON |
| ⚙️ | 模拟设置（速度、单步模式、重置） |
| ▶️ / ⏸️ | 开始 / 暂停模拟 |
| 🌍 | 全局环境设置 |
| ✏️ | 调参工具：点击节点/连线编辑参数 |
| ➡️ | 连线工具：点击两个节点创建捕食关系 |

## 物种参数说明

- **体重 (kg)**：成年个体的平均体重。开启自动调参时，其他参数会根据体重自动计算。
- **初始生物量**：模拟开始时的总生物量。
- **内禀增长率 r**（仅生产者）：理想条件下的日增长率。
- **自然死亡率 d**：每天因年老、疾病等非捕食因素死亡的比例。
- **转化效率 e**（仅消费者）：吃下的食物转化为自身增长的比例，通常 0.05–0.3。
- **种内竞争 c**（仅消费者）：密度越高死亡越快，增强系统稳定性。
- **能量密度 γ**：每克生物量所含能量（kJ/g），植物 ~18，动物 ~22–25。

## 连线参数说明

- **攻击率 a**：捕食者发现并捕获猎物的效率，值越大猎物压力越大。
- **处理时间 h**：消化一单位猎物所需的时间，限制捕食者的最大进食速率。

## 环境参数说明

- **外源输入 I**：每天外界输入的无机营养量（如雨水施肥）。
- **分解速率 ε**：死亡生物量立即矿化返回环境的比例。
- **半饱和浓度 K_N**：生产者生长达到半最大速率时所需的营养浓度。
- **初始营养 N**：模拟起始时的环境营养浓度，提高它可使生产者快速繁殖。

## 模拟控制

- **速度倍数**：1x/2x/5x/10x，加速模拟。
- **单步模式**：开启后每点击一次“单步推进”只模拟一个步长（约0.5天），便于观察细节。
- **重置模拟**：清空所有模拟数据，恢复初始状态。

## 导入/导出

- 点击 💾 下载当前食物网配置（包含物种、连线、环境）为 `.json` 文件。
- 点击 📂 选择之前下载的 JSON 文件，画布和参数会立即恢复，可继续编辑或模拟。

## 物种灭绝

当某物种的个体数低于 0.5 时，判定为灭绝。节点会闪烁并变为灰色，停止参与后续模拟。你可以通过调高初始生物量或改善环境使其复活。

## 小贴士

- 如果系统经常崩溃，尝试为消费者增加**种内竞争系数 c**（如 0.0005），或降低**攻击率 a**，并调高**初始营养 N**。
- 多构建几条食物链可以形成更丰富的动态（例如加入顶级捕食者）。
- 使用单步模式仔细观察生物量变化的每一个转折点。
