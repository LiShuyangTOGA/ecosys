# 🌍 生态系统模拟器 (EcoSys)

一个纯前端的生态系统动态模拟网页应用。你可以**绘制自己的食物网**，调整物种参数和环境条件，然后实时观察种群数量、能量流动和生态金字塔的变化。

> 在线演示：部署后添加链接

## ✨ 功能特性

- 🕸️ **可视化食物网编辑**：拖拽节点、连线创建捕食关系，支持删除和参数调整。
- 📊 **多维度数据展示**：折线图显示个体数/总储能变化，交互式金字塔查看任意时刻的营养结构。
- ⚙️ **灵活的物种参数**：基于体重的自动调参（代谢理论），也可手动精细调节攻击率、处理时间、种内竞争等。
- 🌦️ **动态环境**：可配置营养输入、分解速率、季节波动，模拟不同生态系统。
- 💾 **导入/导出**：将食物网保存为 JSON 文件并可重新加载，方便分享和教学。
- 📚 **内置文档**：点击帮助按钮即可查阅交互式用户手册和科学原理。

## 🚀 快速开始

确保已安装 [Node.js](https://nodejs.org)（版本 ≥ 16）。

```bash
# 克隆仓库
git clone https://github.com/你的用户名/ecosys.git
cd ecosys

# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

浏览器访问 `http://localhost:5173` 即可使用。

### 构建生产版本

```bash
npm run build
```

产出在 `dist/` 目录，可直接部署到任何静态托管服务（如 Cloudflare Pages、GitHub Pages）。

## 🧪 核心模型

基于**多猎物 II 型功能反应**的营养动力学模型，并引入**密度制约竞争**增强稳定性。具体方程参见科学手册。主要特点：

- 物种分为生产者和消费者，消费者可有多条猎物连线。
- 环境包含无机营养池，模拟分解、输入和吸收。
- 自动参数生成基于代谢异速生长理论（体重幂律关系）。
- 灭绝阈值设定为个体数 < 0.5。

## 📝 使用简介

1. 从底部预设栏选择类别，点击物种添加到画布。
2. 用工具栏的 ➡️ 连线工具依次点击猎物和捕食者创建箭头。
3. 用 ✏️ 调参工具点击节点/连线修改参数。
4. 右下角 🌍 设置全局环境。
5. 右上角 ▶️ 开始模拟，节点数字实时变化。
6. 左侧导航栏切换可视化页面（个体数、总储能、金字塔）。

详细操作见用户手册。
